BibleScriptureTagger Creates a hover for Bible references and reveals the verse text and provides a link for further study at the Bible Portal.

== Description ==

Bible Scripture Tagger is a plugin to automatically create a hover for Bible references.
The hover reveals the verse text and provides a link for further study at the Bible Portal website: bibleportal.com.

This plugin can automatically recognize the Bible version. The plugin also provides a simple options page for you to select the default Bible version.

It is very convenient for you to look up the Bible Scripture on internet.

== Installation ==

Easy Install

1. In your WordPress admin, go to Plugins > Add New.
1. Search for BibleScritpureTagger.
1. Click Install, then Install Now, and then Activate Plugin.

Manual Install

1. Download the plugin.
1. Unzip the plugin to your WordPress plugins directory `(/wp-content/plugins/)`.
1. Activate BibleScritpureTagger through the Plugins page in WordPress.

== Usage ==

The plugin works immediately when you activate it on the Plugins page. Customize preferences from Settings. Any changes take effect immediately after clicking Save Changes.
